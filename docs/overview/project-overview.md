# Solana Intent Pilot 项目总览

## 1. 项目定位

Solana Intent Pilot (SIP) 是一个面向 Solana 生态的 Chrome 扩展。它把用户在网页中的自然语言意图、上下文线索与链上执行能力连接起来，让用户无需手动切换 DEX、钱包和分析工具，就可以在浏览器侧边栏里完成“理解行情、评估风险、确认交易”的完整闭环。

项目的核心价值不只是“聊天式下单”，而是构建一个可验证、可扩展、可低延迟执行的 Solana Intent Interface。

## 2. 产品愿景

SIP 希望把浏览器变成 Solana 的智能终端：

- 用户在 Twitter、Birdeye、Dexscreener 等页面发现机会时，无需离开当前页面。
- AI 负责理解模糊表达，生成结构化链上操作意图。
- 本地 Wasm 引擎负责风险校验和高频数据处理，降低对云端黑盒的依赖。
- 执行层负责把意图转成真实可签名的 Solana 交易。

## 3. 目标用户

- 高频浏览链上信息、习惯在社交平台发现交易机会的 Solana 用户
- 希望降低链上操作门槛的普通用户
- 看重速度、安全和上下文感知能力的高级 DeFi 玩家
- 黑客松评审、技术合作方与潜在投资人

## 4. 核心价值主张

### 4.1 意图驱动

用户不需要理解复杂协议参数，只需要表达“我想做什么”。

### 4.2 本地风控

安全校验和敏感分析尽量在浏览器本地完成，形成“云端推理 + 本地验证”的可信执行链。

### 4.3 上下文感知

系统可读取当前标签页上下文、用户选中文本和页面中识别到的代币信息，提高意图解析准确率。

### 4.4 快速执行

通过 Jupiter、Solana Agent Kit 与 RPC 模拟链路，将 AI 建议快速转化为可确认的交易操作。

## 5. 产品范围

### 5.1 MVP 范围

- Chrome Side Panel 常驻交互界面
- 当前页面代币与地址线索感知
- 自然语言到结构化 Intent 的解析
- Jupiter 报价与交易预览
- 本地 Wasm 风险扫描
- 钱包签名与执行结果反馈

### 5.2 后续增强方向

- 多协议支持：Lend、Stake、Transfer、Bridge
- 更细粒度风险评分与交易模拟
- 资产快照、历史行为与偏好建模
- 多节点切换与高可用 RPC 调度
- 实时信号流与策略建议

## 6. 成功标准

- 用户可以在一个页面内完成“发现代币 -> 发出意图 -> 查看风险 -> 签名执行”
- LLM 能稳定输出可校验的结构化 JSON Intent
- 本地风控能在交易确认前给出明确风险反馈
- 交易预览和执行路径足够直观，适合 3 分钟 Demo 展示

## 7. 非目标

- 构建一个全量替代 Phantom 或 Jupiter 的通用钱包/聚合器
- 在首个版本中覆盖所有 Solana DeFi 协议
- 将所有业务逻辑后端化或依赖中心化服务器执行安全判断
